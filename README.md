# 🏡 myProperty – Real Estate Website

A static multi-page real estate website built using HTML and CSS as part of a web development coursework project. The website simulates a property listing platform with structured navigation and user-friendly UI design.

---

## 🚀 Overview

myProperty is a front-end web project designed to demonstrate core web development concepts including:

* Multi-page website structure
* Navigation systems
* UI layout design
* Static content presentation

It represents a basic prototype of a property listing platform.

---

## ✨ Features

* 🏠 Homepage with featured property visuals
* 🧭 Navigation bar linking multiple pages
* 🏢 Dedicated pages for:

  * Apartments for sale
  * Houses for sale
  * Contact page
  * About page
* 🎨 Styled UI using external CSS
* 📱 Structured layout for real estate browsing experience

---

## 🧠 Concepts Used

* HTML5 semantic structure
* CSS styling and layout design
* Multi-page website architecture
* Static UI/UX design principles
* Basic branding and visual hierarchy

---

## ⚙️ Tech Stack

* HTML5
* CSS3
